Chapter 13 :- Html APIs

What is an API ?

English :- API (Application Programming Interface) is a set of built-in features or tools that let HTML/JavaScript interact with the browser.

Hindi :- API एक तरीका होता है जिससे HTML/JavaScript ब्राउज़र के साथ बात कर सकते हैं – जैसे कि यूज़र की लोकेशन लेना, डेटा सेव करना, या एलिमेंट्स को ड्रैग करना।

Popular HTML5 APIs

1 Geolocation API

English :- Purpose: Get the user's current location.

Hindi :- उद्देश्य: यूज़र की वर्तमान location प्राप्त करना।

        
navigator.geolocation.getCurrentPosition(function(position) {
console.log("Latitude: " + position.coords.latitude);
console.log("Longitude: " + position.coords.longitude);
});

Note: ब्राउज़र अनुमति (permission) मांगेगा।

2 Web Storage API

Types :

  • localStorage : डेटा तब तक सेव रहता है जब तक मैन्युअली हटाया न जाए।
  • sessionStorage : डेटा ब्राउज़र टैब बंद होने तक ही रहता है।

localStorage Example:

        
localStorage.setItem("username", "nktech");
let name = localStorage.getItem("username");
console.log(name);

sessionStorage Example:

        
 sessionStorage.setItem("color", "blue");

3 Drag and Drop API

Purpose :- किसी भी एलिमेंट को माउस से खींचकर किसी जगह गिराना।

 
        
<div id="dragme" draggable="true" ondragstart="drag(event)">Drag me</div>
<div id="dropzone" ondrop="drop(event)" ondragover="allowDrop(event)">Drop here</div>

<script>
function allowDrop(ev) {
  ev.preventDefault();
}
function drag(ev) {
  ev.dataTransfer.setData("text", ev.target.id);
}
function drop(ev) {
  ev.preventDefault();
  var data = ev.dataTransfer.getData("text");
  ev.target.appendChild(document.getElementById(data));
}
</script>

4 Canvas API 


        
 <canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

<script>
  let canvas = document.getElementById("myCanvas");
  let ctx = canvas.getContext("2d");
  ctx.fillStyle = "red";
  ctx.fillRect(20, 20, 150, 50);
</script>

5 Fullscreen API 

 
        
function makeFullScreen() {
document.documentElement.requestFullscreen();
}

Complete Example 

 
        
       
<!DOCTYPE html>
<html>
<head>
  <title>HTML5 APIs Example</title>
</head>
<body>

  <h2>Geolocation</h2>
  <button onclick="getLocation()">Get Location</button>
  <p id="location"></p>

  <h2>Local Storage</h2>
  <button onclick="saveData()">Save Name</button>
  <p id="savedData"</p>
 
  <script>
    function getLocation() {
      navigator.geolocation.getCurrentPosition(function(pos) {
        document.getElementById("location").innerText =
          "Lat: " + pos.coords.latitude + ", Long: " + pos.coords.longitude;
      });
    }

    function saveData() {
      localStorage.setItem("name", "NKTechVision");
      document.getElementById("savedData").innerText = localStorage.getItem("name");
    }
  </script>

</body>
</html>
