Chapter 11 :- Block vs Inline Elements

Block vs Inline Elements

What is a Bolck Element?

English :- A block element always starts on a new line and takes the full width available. Block elements are like containers that stack vertically.

Hindi :- Block element हमेशा नई लाइन से शुरू होता है और पूरी चौड़ाई (width) ले लेता है। ये ऐसे होते हैं जैसे एक-एक करके ऊपर-नीचे बॉक्स रखें हो।

Common Block Elements

Block Elements
<div>
<p>
<h1> to <h6>
<section>
article
<header> , <footer>
<form>
<ul> , <ol> , <li>

Block Element Example :

          
 <div>This is a block element</div>
<p>This is a paragraph</p>
<h2>This is a heading</h2>

Explanation :-

English :- Each of these elements appears on a new line and takes full width.

Hindi :- हर एक एलिमेंट नई लाइन से शुरू होगा और पूरी चौड़ाई लेगा।

What is an inline Element?

English :- An inline element does not start on a new line. It only takes as much width as necessary. inline elements stay within a line.

Hindi :- Inline element नई लाइन से शुरू नहीं होता। ये उतनी ही जगह लेता है जितनी जरूरत हो। ये लाइन के अंदर ही रहता है।

Common Inline Elements

Inline Elements
<span>
<a>
<img>
<strong>
<em>
<label>
<input>

Inline Element Example :

          
<p>This is a <span>highlighted</span> word in a sentence.</p>
<a href="#">Click here</a>

Explanation

English :- These elements appear inside a line and do not break the flow.

Hindi :- ये एलिमेंट एक ही लाइन के अंदर रहते हैं और लाइन नहीं तोड़ते।

Feature Block Element Inline Element
Starts on new line Yes No
Takes full width Yes No (only needed width)
Layout Vertical (top to bottom) Horizontal (within text flow)
Example Tags <div> , <p> , <h1> <span> , <a> , <img>

Mixed Example :

        
<div>
  <p>This is a <span style="color:red;">mixed</span> example.</p>
  <a href="#">Inline link</a>
</div>

Explanation :

<div> and <p> are block elements (start from new lines).

<span> and <a> are inline (stay inside the line).

Complete Example : Block vs Inline Elements in HTML

        
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Block vs Inline Elements</title>
</head>
<body>

<h1>Block Element Example</h1>

<div style="background-color: lightblue; padding: 10px;">
This is a block-level <strong>div</strong> element.
</div>

<p style="background-color: lightgreen; padding: 10px;">
This is a block-level <strong>p</strong> paragraph.
</p>

<h2>Inline Element Example</h2>

<p>
This is a sentence with <span style="color: red;">inline</span>
<strong><span>text</span></strong>. Click
<a href="#" style="color: blue;">here</a> for more info.
</p>

<p>
<img src="https://via.placeholder.com/100" alt="Sample" style="vertical-align: middle;">
This <strong>img</strong> tag is also inline.
</p>

</body>
</html>

Important Points

  • Block elements make page layout.
    Block elements पेज का ढांचा बनाते हैं।
  • Inline elements are used for small pieces inside content.
    Inline elements कंटेंट के अंदर छोटे हिस्से के लिए होते हैं।
  • We can style both using CSS.
    दोनों को CSS से स्टाइल किया जा सकता है।
