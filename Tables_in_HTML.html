Chapter 7 :- Tables in Html

Tables in Html

English :- HTML tables are used to display data in rows and columns, just like in MX Excel.

Hindi :- HTML टेबल का उपयोग डेटा को पंक्तियों और स्तंभों (rows & columns) में दिखाने के लिए किया जाता है।

1 Basic Table Structure

        
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>kishor</td>
    <td>22</td>
  </tr>
</table>

Explanation

S.N Tag Use
1 <table> Creates a table
2 <tr> Table row
3 <th> Table heading cell
4 <td> Table data cell

2 Border in Table

English :- border="1" attribute adds visible borders.

Hindi :- border="1" attribute टेबल के चारों ओर बॉर्डर बनाता है।

Example :-

        
<table border="1">
  <tr>
    <th>Item</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Computer</td>
    <td>10,000</td>
  </tr>
</table>

3 Table with Caption

English :- <caption> adds a title on top of table.

Hindi :- <caption> टैग टेबल का शीर्षक दिखाने के लिए उपयोग होता है।

        
<table border="1">
  <caption>Student Data</caption>
  <tr>
    <th>Name</th>
    <th>Roll No</th>
  </tr>
  <tr>
    <td>Amit</td>
    <td>101</td>
  </tr>
</table>

4 Table with colspan and rowspan

Colspan Example :-

English :- colspan="2" means the cell covers 2 columns.

Hindi :- colspan एक सेल को एक से ज़्यादा कॉलम में फैलाने के लिए है।

        
<tr>
  <td colspan="2">Full Row</td>
</tr>

Rowspan Example :-

English :- rowspan="2" means the cell covers 2 rows.

Hindi :- rowspan एक सेल को एक से अधिक रो में फैलाने के लिए होता है।

        
<tr>
  <td rowspan="2">Join Rows<td>
  <td>Row 1</td>
</tr>
<tr>
  <td>Row 2</td>
</tr>

Full Table Example :- 

        
<!DOCTYPE html>
<html>
<head>
  <title>HTML Table by kishor naik</title>
  </head>
<body>
<h2>Student Details</h2>
<table border="1">
  <caption>Class 10 Results</caption>
  <tr>
    <th>Name</th>
    <th>Subject</th>
    <th>Marks</th>
  </tr>
  <tr>
    <td>Kishor</td>
    <td>Math</td>
    <td>95</td>
  </tr>
  <tr>
    <td>Neha</td>
    <td>Science</td>
    <td>90</td>
  </tr>
</table>
</body>
</html>

Quick Summary Table

S.N Tag Use
1 <table> Creates a table
2 <tr> Table row
3 <td> Table data
4 <th> Table heading
5 <border> Adds table border
6 <caption> Table title
7 <colspan> Merge columns
8 <rowspan> Merge rows
