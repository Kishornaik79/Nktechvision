Chapter 10 :- Semantic Elements

What are Semantic Elements ?

English :- Semantic elements clearly describe their meaning to both the browser and the developer.

Hindi :- सेमांटिक एलिमेंट्स अपने अर्थ को ब्राउज़र और डेवलपर दोनों के लिए स्पष्ट रूप से व्यक्त करते हैं।

English :- Non-semantic elements (like <div> and <span>) tell nothing about their content. Semantic elements (like <header>, <article>, <footer>) clearly define the content inside them.

Hindi :- Non-semantic elements (जैसे <div>, <span>) उनके कंटेंट के बारे में कुछ नहीं बताते। Semantic elements (जैसे <header>, <article>, <footer>) कंटेंट का सही अर्थ और भूमिका बताते हैं।

Why Semantic HTML is Important?

  • Improves SEO (Search Engine Optimization).
  • Helps screen readers for accessibility.
  • Makes code easier to read and maintain.
  • SEO (Search Engine Optimization) बेहतर बनाता है।
  • Screen readers के लिए एक्सेसिबिलिटी आसान बनाता है।
  • कोड पढ़ना और maintain करना आसान होता है।

Common Semantic Elements in HTML5

S.N Element Use
1 <header> Represents the header of a page or section.
2 <nav> Defines navigation links.
3 <section> Groups related content together.
4 <article> Represents an independent piece of content.
5 <aside> Defines content aside from the main content.
6 <footer> Represents the footer of a page or section.
7 <main> Main content of the document.
8 <figure> Groups media and captions.
9 <figcaption> Defines a caption for a figure.
10 <mark> Highlights text.
11 <time> Represents date/time information.

Detailed Explanation

1 <header> Element

English :- Defines the top header section of a page or section.

Hindi :- किसी पेज या सेक्शन का शीर्ष (हेडर) भाग दर्शाता है।

Example :

        
 <header>
  <h1>Welcome to My Website</h1>
</header>

Example:

2 <nav> Element

English :- Defines a navigation menu containing links.

Hindi :- एक नेविगेशन मेनू को परिभाषित करता है जिसमें लिंक होते हैं।

Example :

        
<nav>
<a href="#home">Home</a> |
<a href="#services">Services</a> |
<a href="#contact">Contact</a>
</nav>

3 <section> Element

English :- Groups related content into a logical block.

Hindi :- संबंधित कंटेंट को एक लॉजिकल ब्लॉक में ग्रुप करता है।

Example : 

        
<section>
  <h2>Our Services</h2>
  <p>We provide web development services.</p>
</section>

4 <article> Element

English :- Defines independent, self-contained content.

Hindi :स्वतंत्र और आत्मनिर्भर कंटेंट को परिभाषित करता है।

Example : 

        
<article>
  <h2>HTML Tutorial</h2>
  <p>This tutorial helps you learn HTML easily.</p>
</article>

5 <aside> Element

English :- Content related to the main content but placed aside (like ads, links).

Hindi :- मुख्य कंटेंट से संबंधित लेकिन अलग से रखा गया कंटेंट (जैसे विज्ञापन, लिंक)।

Example :

  
<aside>
<h3>Related Articles</h3>
<ul>
  <li><a href="#">CSS Basics</a></li>
  <li><a href="#">JavaScript Guide</a><li>
</ul>
</aside>

6 <footer> Element

English :- Represents the footer area, usually containing copyrights, links.

Hindi :- फूटर क्षेत्र को दर्शाता है, जिसमें आमतौर पर कॉपीराइट या लिंक होते हैं।

Example :

  
<footer>
  <p>Copyright © 2025 NKTechVision</p>
</footer>

7 <main> Element

English :- Defines the main content of the page (unique content).

Example :

  
<main>
  <h1>Main Content Area</h1>
  <p>This is where the primary information appears.</p>
</main>

8 <figure> and <figcaption> Elements

English :- Used to group media (like images) with a caption.

Hindi :- मीडिया (जैसे इमेज) और उसके कैप्शन को एक साथ ग्रुप करने के लिए उपयोग होता है।

Example :

  
<figure>
  <img src="mountains.jpg" alt="Beautiful Mountains">
  <figcaption>View of beautiful mountains during sunrise.</figcaption>
</figure>

9 <mark> Element

English :- Highlights text to draw attention.

Hindi :- टेक्स्ट को हाइलाइट करता है ताकि ध्यान आकर्षित हो।

Example :

  
<p>This is a <mark>very important</mark> message.</p>

10 <time> Element

English :- Represents a specific time or date.

Hindi : एक विशिष्ट समय या तिथि को दर्शाता है।

Example :

  
<p>Event starts at <time datetime="2025-05-01T19:00">7 PM, May 1, 2025</time>.</p>

Complete Practical Example

  
 <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Semantic Elements Example</title>
</head>
<body>

  <header>
    <h1>My Blog</h1>
  </header>

  <nav>
    <a href="#home">Home>/a> |
    <a href="#about">About</a> |
    <a href="#contact">Contact</a>
  </nav>

  <main>
    <section>
      <article>
        <h2>Learning HTML</h2>
        <p>HTML stands for Hyper Text Markup Language.</p>
      </article>
    </section>

    <aside>
      <h3>Related Links</h3>
      <ul>
        <li><a href="#">CSS Basics</a></li;>
        <li><a href="#">JavaScript Guide</a></li;>
      </ul>
    </aside>
  </main>

  <footer>
    <p>© 2025 NKTechVision. All rights reserved.</p>
  </footer>

</body>
</html>

  • Help in SEO and accessibility. SEO और एक्सेसिबिलिटी में मदद करते हैं।
  • Code becomes cleaner and more professional. कोड साफ और ज्यादा प्रोफेशनल बनता है।
