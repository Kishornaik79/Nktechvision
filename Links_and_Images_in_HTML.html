Chapter 4 :- Links and Images

Links and Images in HTML

1 <a> - Anchor Tag (For Links)

English :- This tag is used to create hyperlinks.It uses the href attribute to specify the link address.

Hindi :- यह टैग हाइपरलिंक बनाने के लिए इस्तेमाल होता है। href ऐट्रिब्यूट से लिंक का पता दिया जाता है।

      
  <a href="https://www.google.com">Visit Google</a>

Target Attribute (target="_blank")

English :- Opens the link a new tab.

Hindi :- यह लिंक को नई टैब में खोलने के लिए उपयोग होता है।

      
  <a href="https://www.google.com"target="_blank">Open in new tab</a>

Link to Internal Page

Hindi :- यह अपनी वेबसाइट के अंदर दूसरे पेज पर जाने के लिए लिंक बनाता है।

    
  <a href="about.html">About Us</a>

2 <img> – Image Tag

English :- This tag is used to display an image. It uses src for the image path and alt for alternate text.

Hindi :- यह टैग इमेज दिखाने के लिए इस्तेमाल होता है। src में इमेज का पता और alt में इमेज ना दिखे तो वैकल्पिक टेक्स्ट होता है।

    
  <img src="image.jpg" alt="My Image" width="300" height="200">

Attributes of <img>

Attribute Use
src Source (image file path)
alt Alternate text
width Width of the image
height Height of the image

link on Image

English :- You can wrap an image inside an anchor (<a>) tag to make it clickable.

Hindi :- आप इमेज को <a> टैग के अंदर रखकर उसे क्लिक करने लायक बना सकते हैं।

    
  <a href="https://www.google.com">
  <img src="google.png" alt="Google Logo">
</a>

Example: Links and Images Together

    
  <!DOCTYPE html>
<html>
<head>
  <title>Links and Images by kishor naik</title>
<head>
<body>

<h2>Link Examples</h2>
<p><a href="https://www.google.com">Visit Google</a></p>
<p><a href="about.html" target="_blank">Open About Page</a></p>

<h2>Image Example</h2>
<img src="nature.jpg" alt="Beautiful Nature" width="300">

<h2>Image with Link</h2>
<a href="https://www.wikipedia.org">
  <img src="wiki-logo.png" alt="Wikipedia" width="150">
</a>
</body>
</html>
S.N Tag Use
1 <a> Creates a link
2 href Link address
3 target Open Link in new tab
4 <img> Display an image
5 src Image file path
6 alt Text if image not shown
7 width Image width
8 height Image height
