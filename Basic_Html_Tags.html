Chapter 2 :- Basic Html Tags

Basic Html Tags

1 < !DOCTYPE html > - Doctype Declaration

English :- This tag tells the browser that the document is written in HTML5.

Hindi :- यह टैग ब्राउज़र को बताता है कि यह दस्तावेज़ HTML5 में लिखा गया है।

        
         <!DOCTYPE html>

2 <html> - Root Element

English:-This is the main tag that contains the whole HTML document.

Hindi:- यह मुख्य टैग होता है जो पूरे HTML दस्तावेज़ को अपने अंदर रखता है।

          
        <html>...</html>

3 <head> - Head Section

English :- This section contains information like title,meta tags, and links. It is not visible on the page.

Hindi :-यह सेक्शन शीर्षक, मेटा टैग्स और लिंक जैसी जानकारी रखता है। यह वेब पेज पर दिखाई नहीं देता।

          
       <head>...</head>

4 <title> - Page Title

English :- This tag sets the title shown on the browser tab.

Hindi :- यह टैग ब्राउज़र टैब में दिखाई देने वाला शीर्षक सेट करता है। 

        
  <title>My Website</title>

5 <body> - Body Section

English :- This contains everything visible on the web page like text, images, links, etc.

Hindi :- यह टैग उन सभी चीज़ों को रखता है जो वेब पेज पर दिखाई देती हैं, जैसे कि टेक्स्ट, इमेज, लिंक आदि।

        
      <body>...</body>

6 <h1> to <h6> - Headings

English :- There are six levels of headings. <h1> is the biggest and <h6> is the smallest.

Hindi :- हेडिंग के छह स्तर होते हैं। <h1> सबसे बड़ी होती है और <h6> सबसे छोटी।

        
  <h1> Main Heading </h1>
  <h2> Smallest Heading </h2>

7 <p> - Paragraph

English :- This tag is used to define a paragraph of text.

Hindi :- यह टैग टेक्स्ट का एक अनुच्छेद बनाने के लिए उपयोग किया जाता है।

        
    <p>This is a paragraph.</p>

8 <br> - Line Break

English :-This tag breaks the line and moves the text to the next line.

Hindi :- यह टैग लाइन को तोड़कर टेक्स्ट को अगली पंक्ति में ले जाता है।

        
          Line one <br> Line two

9 <hr> - Horizontal Line

English :- This is creates a horizontal line across the page.

Hindi :- यह टैग वेब पेज पर एक क्षैतिज रेखा बनाता है।

        
        <hr>

10 <!--Comment-->- Comment Tag

English :- This tag adds comment in the code which is not shown on the web page.

Hindi :- यह टैग कोड में टिप्पणी जोड़ता है जो वेब पेज पर दिखाई नहीं देती।

        
    <!--This is a comment-->

Full Example Code :-

        
  <!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is a paragraph.</p>
  <hr>
  <p>Line One<br>Line Two</p>
  <!-- This is a comment -->
</body>
</html>
