Chapter 8 :- Forms in html

Forms in HTML

What is a Form ?

English :- An HTML form is used to collect user input.

Hindi :- HTML Form का उपयोग यूज़र से जानकारी (input) लेने के लिए किया जाता है।

Forms के अंदर हम Text Box, Radio Button, Checkbox, Submit Button वगैरह यूज़ करते हैं।

Basic Form Structure

          
<form action="submit.php" method="post">
  <!-- Form elements go here -->
</form>
  • action = Where the form data will go after submission.
    action: फॉर्म जमा करने के बाद डेटा कहां भेजना है।
  • method = How the data will be sent (GET या POST).
    method: डेटा कैसे भेजा जाएगा (GET या POST).

Common Form Elements with Example

1 <input> (Text Field)

English :- Creates a textbox for entering name.

Hindi :- नाम डालने के लिए एक टेक्स्ट बॉक्स बनाता है।

        
<form> 
  <label for="name">Name:</label>
  <input type="text" id="name" name="username">
</form>

2 <input type="password"> (Password Field)

English :- Hides the entered text as dots.

Hindi :- पासवर्ड डालते समय अक्षर डॉट्स में बदल जाते हैं।

        
<form>
  <label for="password">Password:</label>
  <input type="password" id="password" name="userpassword">
</form>

3 <input type="email"> (Email Field)

English :- Validates email format.

Hindi :- सही ईमेल फॉर्मेट की जांच करता है।

        
<form>
  <label for="email">Email:<label>
  <input type="email" id="email" name="useremail">
</form>

4 <input type="radio"> (Radio Button)

English :- Select one option among many.

Hindi :- कई विकल्पों में से एक विकल्प चुनने के लिए।

        
<form>
  Gender:<br>
  <input type="radio" id="male" name="gender" value="male">
  <label for="male">Male<label><br>
  
  <input type="radio" id="female" name="gender" value="female">
  <label for="female">Female<label>
</form>

5 <input type="checkbox"> (checkbox)

English :- You can select multiple hobbies.

Hindi :- आप कई शौक एक साथ चुन सकते हैं।

        
<form> 
  Choose your hobbies:<br>
  <input type="checkbox" id="hobby1" name="hobby" value="Reading">
  <label for="hobby1">Reading</label><br>

  <input type="checkbox" id="hobby2" name="hobby" value="Traveling">
  <label for="hobby2">Traveling<label> <br>

  <input type="checkbox" id="hobby3" name="hobby" value="Coding">
  <label for="hobby3">Coding<label>
</form>

6 <textarea> (Multi-Line Text Input)

English :- For typing large messages.

Hindi :- बड़े संदेश या फीडबैक के लिए।

        
<form>
  <label for="message">Your Message:</label><br>
  <textarea id="message" name="usermessage" rows="4" cols="30"></textarea>
</form>

7 <select> and <option> (Dropdown List)

English :- Dropdown menu to select one country.

Hindi :- एक देश चुनने के लिए ड्रॉपडाउन मेनू।

       
<form>
  <label for="country">Choose Country:</label>
  <select id="country" name="usercountry">
    <option value="India">India</option>
    <option value="USA">USA</option>
    <option value="Australia">Australia</option>
  </select>
</form>

8 <button> ( Button to Submit or Click)

English :- Button to submit form.

Hindi :- फॉर्म सबमिट करने के लिए बटन।

       
<form> 
  <button type="submit">Submit Form</button>
</form>

Complete Practical Example :- 

        
<!DOCTYPE html> 
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Registration Form</title>
</head>
<body>

  <h2>User Registration</h2>

  <form action="submit.php" method="post">

    <label for="fullname">Full Name:</label><br>
    <input type="text" id="fullname" name="fullname" required><br><br>

    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email" required><br><br>

    <label for="password">Password:</label><br>
    <input type="password" id="password" name="password" required><br><br>

    <br>Gender:</p>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>

    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br><br>

    <p>Hobbies:</p>
    <input type="checkbox" id="reading" name="hobby" value="Reading">
    <label for="reading">Reading</label>

    <input type="checkbox" id="coding" name="hobby" value="Coding">
    <label for="coding">Coding</label><br><br>

    <label for="country">Country:</label><br>
    <select id="country" name="country">
      <option value="India">India</option>
      <option value="USA">USA</option>
    </select><br><br>

    <label for="message">Message:</label><br>
    <textarea id="message" name="message" rows="4" cols="30"></textarea><br><br>

    <button type="submit">Register</button>

  </form>
</body>
</html>

Important Points

  • Always use label with input for better accessibility.
    हमेशा input के साथ label का उपयोग करें।
  • required attribute से फील्ड को अनिवार्य बना सकते हैं।
    required attribute से फॉर्म भरना अनिवार्य होता है।
  • name attribute बहुत जरूरी है ताकि डेटा सर्वर पर सही भेजा जाए।
    name attribute form के डेटा को सर्वर पर भेजने के लिए जरूरी होता है।
