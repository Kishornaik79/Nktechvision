Chapter 4 :- Attributes in html

what are Attributes ?

English :- html attributes provide extra information about html elements.
They are always written inside the opening tag.

Hindi :- HTML में attributes किसी टैग के बारे में अतिरिक्त जानकारी देते हैं। यह हमेशा opening tag के अंदर लिखे जाते हैं।

Structure :- 

        
 <tagname attribute="value">Content</tagname>

Example :- 

        
 <a href="https://www.google.com">Google</>

यहां href attribute है, और इसका मान (value) "https://www.google.com" है।

Most Common Html Attributes

Attribute Element Use
href <a> Link address
src <img> Image source
alt <img> Alternate text if image fails
title All Tooltip on hover
width <img> , <table> Width of element
height <img>, <table> Height of element
style All Inline CSS styling
id All Unique identifier
class All Group identifier for styling

Examples :- of Attributes

1 href Attribute

        
 <a href="https://www.example.com">Visit another site</a>

English :- This link goes to another website.

Hindi :- यह लिंक किसी दूसरी वेबसाइट पर ले जाता है।

2 src and alt Attributes

        
 <img src="flower.jpg" alt="Beautiful Flower">

English :- if the image doesn't load, "Beautiful Flower" will show.

Hindi :- Hindi: अगर इमेज नहीं लोड हुई, तो "Beautiful Flower" लिखा आएगा।

3 style Attribute (Inline CSS)

        
 <p style="color: red;">This is red text </p>

English :- This paragraph will appear in red color.

Hindi :- यह पैराग्राफ लाल रंग में दिखाई देगा।

4 title Attribute 

        
 <p title="This is a tooltip">Hover over me!</p>

English :- A message will show when you hover over the text.

Hindi :- जब आप माउस ले जाएंगे, तो एक छोटा मैसेज दिखाई देगा।

5 id Unique Identifier

        
 <h2 id="about">About Me</h2>

English :- Used to target this element specifically (once only).

Hindi :- किसी एक खास एलिमेंट को पहचानने के लिए।

6 class Group Identifier 

      
<p class="info">My name is Rahul</p>
<p class="info">I am learning HTML</p>

English :- Use the same class for similar elements.

Hindi :-एक जैसे एलिमेंट्स को ग्रुप करने के लिए।

Complete Code Using All Attributes:

        
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>HTML Attributes Example</title>
</head>
<body>

  <h1 id="main-heading" title="Welcome!">Welcome to My Webpage</h1>

  <img src="profile.jpg" alt="My Profile Photo" class="profile-image" style="width: 200px; border-radius: 10px;">

  <p class="info" style="color: red;" title="This is an intro paragraph">
    Hello, my name is <span class="highlight">kishor naik</span>. I am learning HTML.
  </p>

  <a href="https://www.google.com" target="_blank" title="Go to Google" class="link-button">
    Visit Google
  </a>

  <h2 id="about-section">About Me<h2>
  <p class="info">I love coding and designing web pages.</p>

</body>
</html>
