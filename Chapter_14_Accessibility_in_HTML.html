Chapter 14 :- Accessibility

What is Accessibility ?

English :- Accessibility means designing websites in a way that everyone, including people with disabilities, can easily use and understand them.

Hindi :- Accessibility का मतलब है वेबसाइट को इस तरह बनाना कि विकलांग (disabled) व्यक्ति भी उसे आसानी से पढ़ और समझ सके।

Why is Accessibility Important ?

  • Better user experience for everyone सभी यूज़र्स को अच्छा अनुभव मिलता है।
  • Legal Requirement in many countries कई देशों में यह कानून से जरूरी होता है।
  • Helps with SEO (Search Engine Optimization) गूगल आपकी वेबसाइट को बेहतर समझ पाता है।

Key Features for html Accessibility

1 Use Semantic html Tags

Semantic tags जैसे <header>, <nav>, <main>, <footer> से स्क्रीन रीडर को साइट की संरचना समझने में मदद मिलती है।

Example:

        
<header>  
  <h1>Welcome to My Website</h1>
</header>

Alt Text Images

alt attribute screen readers को बताता है कि इमेज में क्या है।

        
<img src="logo.png" alt="Company Logo">

English :- Describes image for screen readers.

Hindi :- स्क्रीन रीडर के लिए इमेज का वर्णन करता है।

3 Label for Form Inputs

Form controls को पहचानने के लिए <label> जरूरी है।

      
<label for="email">Email:</label>
<input type="email" id="email" name="email">

English :- Associates label with input.

Hindi :- इनपुट फील्ड के साथ लेबल को जोड़ता है।

4 Use ARIA Attributes

ARIA (Accessible Rich Internet Applications) attributes help when default html elements are not enough.

      
<div role="alert">Form submission failed!</div>

English :- Screen readers will treat it as an alert.

Hindi :- स्क्रीन रीडर इसे चेतावनी के रूप में पढ़ेगा।

5 Keyboard Navigation Support

सुनिश्चित करें कि यूज़र केवल कीबोर्ड से आपकी साइट पर नेविगेट कर सकें।

  • Use tabindex="0" to make an element focusable.
  • Avoid removing outline in CSS.

6 Use Proper Headings Heading Tags

Headings structure (<h1> to <h6>)

        
<h1>Main Title</h2>
<h2>Sub Title</h2>

Complete Accessible Form Example

   
<form> <!-- edid code from here-->
<label for="name">Name:</label>
<input type="text" id="name" name="name" required>

<label for="email">Email:</label>
<input type="email" id="email" name="email" required>

<button type="submit">Submit</button>
</form>
