Chapter 1 :- Introduction to HTML

1. Introduction to HTML

1.1 What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard language used to create web pages and web applications.

HTML kay karta hai?

HTML web page ke content ka structure banata hai.Ye batata hai ki web page par kya dikhana hai - jaise text,images,links,videos,buttons,etc.

Example:


      <!DOCTYPE html>
      <html>
      <head>
         <title> My First page </title>
      </head>
      <body>
         <h1> Hello World! </h1>
         <p> This is my first HTML page </p>
      </body>
      </html>

1.2 Full Form of HTML

H-Hyper T-Text M-Markup L-Language

HyperText: means linking text together.

Markup Language: means using tags to define elements in a document.

1.3 History of HTML

Version Year Released Key Features
HTML 1.0 1993 Basic structure
HTML 2.0 1995 Forms added
HTML 3.2 1997 Tables,Scripts
HTML 4.01 1999 CSS support
HTML 5 2014 Multimedia,Mobile-friendly,APIs

Feature HTML HTML5

Mobile support No Yes
Offline Storage No Yes (LocalStorage)

1.5 How Browsers Read HTML?

Web browsers like Chrome, Firefox, and Edge read your HTML code and display it as a web page.

Browser HTML file ko line by line read karta hai aur uska output screen par dikhata hai.

1.6 Structure of an HTML Document

Basic HTML document ka structure kuch is tarah hota hai: 

  
   <!DOCTYPE html>  <!-- HTML5 declaration -->
<html> <!-- Root element -->
  <head> <!-- Metadata -->
    <title>Title</title> <!-- Title of the page -->
  </head>
  <body> <!-- Visible content -->
    <h1>Main Heading</h1>
     <p>This is a paragraph.</p>
  </body>
 </html>
