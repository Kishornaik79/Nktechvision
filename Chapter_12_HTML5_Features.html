Chapter 12 :- HTML5 Features

What is HTML5 ?

English :- HTML5 is the latest version of html that introduced many new features to build modern websites and web applications.

Hindi :- HTML5 HTML का नया वर्शन है जिसमें कई नए टैग्स और फीचर्स जोड़े गए हैं जिससे वेबसाइट बनाना आसान और ज़्यादा फंक्शनल हो गया है।

Top HTML5 Features

1 New Semantic Elements

English :- Meaningful tags to structure content clearly.

Hindi :- साइट के भागों को सही ढंग से समझाने वाले नए टैग।

Examples:

        
<header>, <footer>, <nav>, <section>, <article>, <aside>

2 Audio and Video Support

English :- No need for Flash. You can directly play media files.

Hindi :- अब फ्लैश की जरूरत नहीं, HTML से ही ऑडियो/वीडियो चला सकते हैं।

Examples:

        
<audio controls src="song.mp3"></audio>
<video controls src="video.mp4"></video>

3 Canvas Element

English :- Used to draw graphics using JavaScript (like games, charts, animations).

Hindi :- JavaScript की मदद से graphics बनाने के लिए।

Example:

        
<canvas id="myCanvas" width="200" height="100"></canvas>

4 SVG (Scalable Vector Graphics)

English :- For drawing 2D graphics that scale without losing quality.

Hindi :- इमेज को बिना धुंधला किए स्केल करने के लिए।

Example:

        
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="blue" />
</svg>

5 New Input Types

English :- Better and mobile-friendly form controls.

Hindi :- नई इनपुट टाइप्स से फॉर्म यूज़र-फ्रेंडली बनते हैं।

Examples:

        
<input type="email">
<input type="date">
<input type="range">
<input type="color">

6 Placeholder Attribute

English :- Shows hint text inside inputs.

Hindi :- इनपुट बॉक्स में संकेत देने के लिए।

Example:

        
<input type="text" placeholder="Enter your name">

7 Local Storage and Session Storage

English :- Store data in browser without database.

Hindi :- ब्राउज़र में डेटा सेव करने की सुविधा।

Example:

        
 localStorage.setItem("username", "nktech");

Geolocation API

English :- Get user's location using browser.

Hindi :- यूज़र की लोकेशन ट्रैक करने के लिए।

Example:

        
navigator.geolocation.getCurrentPosition(showPosition);

9 Responsive Design Support

English :- Better mobile view support using <meta> tag.

Hindi :- मोबाइल पर बेहतर दिखने के लिए।

Example:

        
<meta name="viewport" content="width=device-width, initial-scale=1.0">

10 Drag and Drop Support

English :- You can drag items on a page without JavaScript libraries.

Hindi :- HTML5 से ड्रैग और ड्रॉप आसानी से हो सकता है।

Complete Code Example:

        
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>HTML5 Features</title>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>

<header><h1>Welcome to HTML5</h1></header>

<section>
<article>
<h2>Video Example</h2>
<video controls width="300">
<source src="video.mp4" type="video/mp4">
</video>
</article>

<article>
<h2>Canvas Example</h2>
<canvas width="200" height="100" style="border:1px solid black;"></canvas>
</article>

<article>
<h2>Form Input Types</h2>
<form>
<input type="email" placeholder="Enter email"><br>
<input type="date"><br>
<input type="color"><br>
</form>
</article>
</section>

<footer>&copy; 2025 NKTechVision</footer>

</body>
</html>
