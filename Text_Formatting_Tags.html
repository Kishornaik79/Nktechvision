Chapter 3 :- Text Formatting Tags

Text Formatting Tags

1 <b> - Bold Text

English :- This tag makes the text bold (strong appearance).

Hindi :- यह टैग टेक्स्ट को मोटा (बोल्ड) बनाता है।

        
  <b>This is bold text.</b>

2. <strong> – Strong Importance (Also Bold)

English :- This also makes text bold, but it shows strong importance in meaning (for SEO and accessibility).

Hindi :- यह टैग टेक्स्ट को बोल्ड बनाता है, लेकिन इसका अर्थ होता है "महत्वपूर्ण जानकारी"।

      
  <strong>Important information.</strong>

3. <i>- Italic Text

English :- This tag makes the text italic (slanted)

Hindi :- यह टैग टेक्स्ट को तिरछा बनाता है।

        
  <i>This is italic text.</i>

4 <em> - Emphasized Text (Also Italic)

English :- This tag also makes the text italic, but it gives emphasis (stress) to the words.

Hindi :- यह टैग शब्दों पर ज़ोर देने के लिए उपयोग किया जाता है और टेक्स्ट को तिरछा बनाता है।

        
  <em>Very important </em>

5 <u> - Underlined Text

English :- This tag underlines the text.

Hindi :- यह टैग टेक्स्ट के नीचे रेखा (underline) खींचता है।

        
  <u>This is underlined text. </u>

6 <mark> - Highlighted Text

English :- This tag highlights the text with a yellow background

Hindi :-यह टैग टेक्स्ट को पीले रंग से हाइलाइट करता है।

        
  <mark>Highlight this text  </mark>

7 <small> - Smaller Text

English :- This tag makes the text smaller than normal.

Hindi :- यह टैग टेक्स्ट को छोटा दिखाता है।

        
  <small>this is small text. </small>

8 <del> - Deleted Text

English :- This tag shows the text as deleted (with a line through it).

Hindi :- यह टैग टेक्स्ट को ऐसा दिखाता है जैसे उसे काट (डिलीट) दिया गया हो।

        
  <del>this text is deleted. </del>

9 <ins> - Inserted Text

English :- This tag shows inserted text with an underline.

Hindi :- यह टैग नया जोड़ा गया टेक्स्ट दिखाने के लिए उपयोग होता है।

        
  <ins>this is newly added. </ins>

10 <sub> - Subscript

English :- This tag is used to write text slightly below the normal line (like H₂O).

Hindi :- यह टैग टेक्स्ट को नीचे की ओर छोटा करके दिखाता है, जैसे कि रसायन सूत्रों में।

        
  H<sub>2</sub>O

11 <sup> - Superscript Text

English :- This tag is used to write text slightly above the normal line (like x²).

Hindi :- यह टैग टेक्स्ट को ऊपर की ओर छोटा करके दिखाता है, जैसे कि गणितीय सूचकों में।

        
  X<sup>2</sup>
