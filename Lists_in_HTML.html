Chapter 5 :- Lists in Html

Lists in html

Html provides 3 types of lists

1. Ordered List – क्रमबद्ध सूची (जैसे 1, 2, 3...)

2. Unordered List – बिना क्रम की सूची (जैसे ●, ●, ●...)

3. Description List – विवरण सूची (जैसे शब्द और उसका अर्थ)

1 <ol>- Ordered List

English :- This tag creates a list with numbers or letters.

Hindi :- यह टैग क्रमबद्ध (जैसे 1, 2, 3) सूची बनाता है।

        
 <ol>
  <li>ENGLISH</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>PYTHON</li>
  <li>JAVA</li>
</ol>

2 <ul> - Unordered List

English :- This tag creates a list with bullet points.

Hindi :- यह टैग बिना क्रम वाली (● ● ●) बुलेट लिस्ट बनाता है।

        
 <ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Mango</li>
</ul>

3 <li> - List Item

English :- This tag is used inside <ol> or <ul> to define each item in the list.

Hindi :- यह टैग किसी सूची का एक आइटम दिखाने के लिए होता है।

        
 <li>Item Name</li>

4 <dl> -

English :- This tag is used to create a list of terms and their description.

Hindi :- यह टैग शब्द और उनके विवरण के लिए प्रयोग किया जाता है।

        
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
S.N Tag Use
1 <ol> Ordered List
2 <ul> Unordered List
3 <li> List Item
4 <dl> Description List
5 <dt> Definition Term
6 <dd> Definition Description

Example :- All Lists Together

        
<!DOCTYPE html>
<html>
<head>
  <title>Chapter 5 - HTML Lists</title>
</head>
<body>

<h2>Ordered List</h2>
<ol>
  <li>Wake up</li>
  <li>Brush Teeth</li>
  <li>Go to Work</li>
</ol>

<h2>Unordered List</h2>
<ul>
  <li>Pen</li>
  <li>Pencil</li>
  <li>Eraser</li>
</ul>

<h2>Description List</h2>
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>JS</dt>
  <dd>JavaScript</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
</body>
</html>

Quick Summary

Ordered List (<ol>) – नंबर वाली लिस्ट

Unordered List (<ul>) – बुलेट वाली लिस्ट

Description List (<dl>) – शब्द और उसका मतलब
